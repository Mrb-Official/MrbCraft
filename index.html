What is MrbCraft?

MrbCraft is a beautifully crafted Linux distribution based on Debian. Built with a pre-configured Openbox window manager and macOS-style dock, it provides a stunning, developer-focused out-of-the-box experience.

Download ISO (3.6 GB)

Features

Bloat-Free Skeleton

MrbCraft stays clean and secure. Built with a specialized skeleton structure, it provides a fast, zero-bloat environment without exposing personal data.

Stunning UI

With pre-configured Plank/Polybar docks, sleek dark mode aesthetics, and the beautiful Desert-SDDM login screen, MrbCraft looks absolutely amazing.

Developer Ready

Engineered for modern workflows. With rock-solid Debian stability, it is the perfect environment for coding, web development, and terminal lovers.
